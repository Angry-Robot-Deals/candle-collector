# Memory Bank Docs

This folder holds documentation **about** the Memory Bank system usage in this project.

- Task docs and reports live in `memory-bank/tasks/`, `memory-bank/qa/`, `memory-bank/reports/` etc.
- See project root `.cursor/rules` and `memory-bank-paths.mdc` for paths and conventions.
- Do not put secrets (passwords, keys, real URLs) in any Memory Bank files; the repository is public.
